Periodic Table

The Periodic table is a list of all the known chemical elements. Currently, there are 118 different elements in the Periodic table. Elements are the main building blocks of all matter.

Chemical Symbols

Each element has a chemical symbol that contains one or two letters, which is a shortcut for describing that element. The chemical symbol for the element Hydrogen is H and the symbol for Yttrium is Y. The elements are listed in the Periodic table is order of increasing Atomic Number.
