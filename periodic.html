Periodic Table

The periodic table is a table of the chemical elements that the elements are arranged by order of the atomic number. The standard form of the table contains of a grid of elements that have 18 columns and 7 rows. The columns of the table are called groups and the rows of the table are called periods. The table can be used to determine the relationships between the properties of the elements. Scientists use the periodic table to analyze chemical behavior.
